CentOS and RHEL 6.0: semanage, command not found?

By xsa On September 14, 2011 · Leave a Comment

Are you a CentOS or RHEL 6.0 administrator trying to manage seLinux but have found the command line tools aren’t available?

 Bash |  copy code |? 
1
-bash: semanage: command not found

This is common when you customize the default package list during an install. Don’t worry, though; it’s easily resolved.

 Bash |  copy code |? 
1
yum -y install policycoreutils-python

.. and you are good to go. Good luck with seLinux (because you will probably need it!)
