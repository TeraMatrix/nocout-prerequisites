This would interest me as well, but i can give you my "solution":

We are running two identical servers and installed the OMD on a shared NFS partition.
Once a server fails we can manually just start the site on the other host.

Monitoring the active server works either via a minimalist site that only checks the active host via the agent omd check or via a cronjob which sends out an email once the main server fails.

Failover is manual but so far we had no problem with this setup
